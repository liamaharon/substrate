Compute VRF pre-output paired with input from secret key and some VRF input, like a Transcript or a VrfInput.

As the arkworks hash-to-curve infrastructure looks complex, we employ arkworks’ simpler UniformRand here, which uses shitty try and increment. We strongly recommend you use a better hash-to-curve manually.
